API 文档

在 Cloudflare Dashboard 部署 Worker 并绑定 Workers AI 后，于环境变量中设置 API_KEY；请求时携带 Authorization: Bearer <API_KEY>。

路由

  • GET /health — 健康检查
  • GET /models — 模型列表，支持 ?type=text-generation
  • POST /generate — 生成（需 Token）

POST /generate

请求体 JSON：

  • prompt (string, 必填) — 输入提示词
  • model (string) — 模型 ID，默认按 type 选取
  • type — text-to-image | text-generation | …
  • stream (boolean) — 文本流式输出
  • width / height (number, 可选) — 文生图尺寸，仅 FLUX.2 生效，默认 1024
curl -X POST ORIGIN/generate \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"a cat","type":"text-to-image"}' \
  -o out.jpg
curl -X POST ORIGIN/generate \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"explain react hooks in markdown","type":"text-generation"}'

FLUX.2 文生图

FLUX.2 系列（Klein 4B / Klein 9B / Dev）在 Workers AI 侧要求 multipart/form-data，不能直接传 { prompt }。 本 API 仍统一使用 JSON 调用，Worker 会自动转换为 multipart（prompt、width、height）。

  • @cf/black-forest-labs/flux-2-klein-4b
  • @cf/black-forest-labs/flux-2-klein-9b
  • @cf/black-forest-labs/flux-2-dev

FLUX.1 Schnell 等其它文生图模型仍使用普通 JSON prompt 即可。

curl -X POST ORIGIN/generate \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"a cat on the beach","type":"text-to-image","model":"@cf/black-forest-labs/flux-2-klein-4b","width":1024,"height":1024}' \
  -o out.png

模型

完整目录见 Cloudflare Workers AI Models。本项目收录 80 个官方模型。

国内访问

*.workers.dev 在国内可能较慢或不稳定。推荐在 Cloudflare Dashboard 为 Worker 绑定自定义域名（域名需接入 Cloudflare DNS，保持橙色云朵代理）。步骤参考 cf-proxy 部署指南。

  • 本地 dev pending：Workers AI 需连接 Cloudflare，国内网络易超时；请 npm run deploy 后用线上地址调试 /generate
  • 无法访问 workers.dev：可部署 cf-proxy，通过 /proxy/<你的-worker>.workers.dev/... 转发
  • 静态资源：字体与样式已本地化，不依赖 Google CDN
curl https://YOUR-PROXY.workers.dev/proxy/YOUR-API.workers.dev/health
