API 文档

Worker 只读请求头：Authorization: Bearer <Token>（必填）、CF-Account-Id（AI 接口必填）。Demo 右上角 input 填写后自动放入 Header。

凭证

  • Authorization: Bearer <Token> — Cloudflare API Token，需 Workers AI 权限
  • CF-Account-Id: <Account ID> — 32 位 hex；GET /v1/models 可不传，生成类接口必填

Account ID 获取：Dashboard → Workers AI → 右侧 Use REST API 复制（非纯数字）。

路由（OpenAI 兼容）

  • GET /health — 健康检查
  • GET /v1/models — 模型列表，支持 ?type=text-generation
  • POST /v1/chat/completions — 对话补全，支持 stream: true SSE
  • POST /v1/responses — Responses API，支持 stream: true
  • POST /v1/images/generations — 文生图，返回 b64_json

POST /v1/chat/completions

与 OpenAI Chat Completions 兼容。Workers AI 原生 SSE 会在网关层转换为 OpenAI chunk 格式。

  • model — Workers AI 模型 ID，如 @cf/meta/llama-3.1-8b-instruct-fast
  • messages — [{ role, content }] 数组
  • stream — true 时返回 text/event-stream
  • temperature / max_tokens — 可选
curl -N -X POST ORIGIN/v1/chat/completions \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "CF-Account-Id: YOUR_ACCOUNT_ID" \
  -H "Content-Type: application/json" \
  -d '{"model":"@cf/meta/llama-3.1-8b-instruct-fast","messages":[{"role":"user","content":"hello"}],"stream":true}'

POST /v1/responses

与 OpenAI Responses API 兼容。流式时发送 response.output_text.delta 等事件。

curl -N -X POST ORIGIN/v1/responses \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "CF-Account-Id: YOUR_ACCOUNT_ID" \
  -H "Content-Type: application/json" \
  -d '{"model":"@cf/moonshotai/kimi-k2.7-code","input":"你好","stream":true}'

POST /v1/images/generations

与 OpenAI Images API 兼容。FLUX.2 模型会自动走 multipart，客户端仍用 JSON。

  • prompt — 必填
  • model — 文生图模型 ID
  • size — 如 1024x1024（FLUX.2 生效）
curl -X POST ORIGIN/v1/images/generations \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "CF-Account-Id: YOUR_ACCOUNT_ID" \
  -H "Content-Type: application/json" \
  -d '{"model":"@cf/black-forest-labs/flux-1-schnell","prompt":"a cat"}' \
  | jq -r '.data[0].b64_json' | base64 -d > out.jpg
curl -X POST ORIGIN/v1/images/generations \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "CF-Account-Id: YOUR_ACCOUNT_ID" \
  -H "Content-Type: application/json" \
  -d '{"model":"@cf/black-forest-labs/flux-2-klein-4b","prompt":"a cat on the beach","size":"1024x1024"}' \
  | jq -r '.data[0].b64_json' | base64 -d > out.png

SDK 接入示例

OpenAI SDK 需自定义 fetch 传入 CF-Account-Id：

import OpenAI from "openai";

const accountId = process.env.CF_ACCOUNT_ID;
const client = new OpenAI({
  apiKey: process.env.CF_API_TOKEN,
  baseURL: "https://YOUR-WORKER.workers.dev/v1",
  fetch: (url, init) =>
    fetch(url, {
      ...init,
      headers: {
        ...Object.fromEntries(new Headers(init?.headers)),
        "CF-Account-Id": accountId,
      },
    }),
});

const stream = await client.chat.completions.create({
  model: "@cf/moonshotai/kimi-k2.7-code",
  messages: [{ role: "user", content: "hello" }],
  stream: true,
});

for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0]?.delta?.content ?? "");
}

模型

完整目录见 Cloudflare Workers AI Models。本项目收录 80 个官方模型。响应含 cf_type、cf_name、cf_description 扩展字段；Demo 页使用 GET /v1/models?type=... 筛选。

国内访问

*.workers.dev 在国内可能较慢或不稳定。推荐在 Cloudflare Dashboard 为 Worker 绑定自定义域名（域名需接入 Cloudflare DNS，保持橙色云朵代理）。步骤参考 cf-proxy 部署指南。

  • 本地 dev pending：Workers AI 需连接 Cloudflare，国内网络易超时；请 npm run deploy 后用线上地址调试
  • 无法访问 workers.dev：可部署 cf-proxy，通过 /proxy/<你的-worker>.workers.dev/v1/... 转发
  • 静态资源：字体与样式已本地化，不依赖 Google CDN
curl https://YOUR-PROXY.workers.dev/proxy/YOUR-API.workers.dev/health
